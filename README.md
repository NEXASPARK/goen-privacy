# goen（ご縁）プライバシーポリシー

株式会社NEXASPARK が提供するアプリ「goen（ご縁）」のプライバシーポリシー公開用リポジトリです。

公開URL: https://nexaspark.github.io/goen-privacy/

## 多言語

日本語を基準言語とし、英語・中国語（簡体字）・韓国語に対応しています。各ページ上部の言語スイッチャーで切り替えできます。齟齬がある場合は日本語版が優先します。

| 言語 | パス | URL |
|---|---|---|
| 日本語（基準） | `index.html` | https://nexaspark.github.io/goen-privacy/ |
| English | `en/index.html` | https://nexaspark.github.io/goen-privacy/en/ |
| 简体中文 | `zh-Hans/index.html` | https://nexaspark.github.io/goen-privacy/zh-Hans/ |
| 한국어 | `ko/index.html` | https://nexaspark.github.io/goen-privacy/ko/ |

※ 本文の編集は各言語の `index.html` を更新してください。ソースは本アプリのリポジトリ `goen` の `legal/privacy/` で管理しています。
